## 🚀 Começando


Clone o projeto e acesse a pasta

```bash
https://github.com/JoaoVlltor/moveit-next && cd moveit-next
```

Siga os passos abaixo
```bash
# Instale as dependências
$ yarn

# 
Comece o projeto
$ yarn dev
```
